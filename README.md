# CommonTemplate

#### 介绍

基于 vue2.X 的前端通用模板，包含通用工具类，axios 封装，基础 webpack 配置等

#### 软件架构

基于 vue 2.x 的前端通用代码

#### 安装教程

1.  git clone 'https://gitee.com/rawcloud/common-template.git'
2.  cd common-template
3.  npm i 或 cnpm i

#### 使用说明

1.  npm run serve
2.  npm run bulid
3.  npm run lint
